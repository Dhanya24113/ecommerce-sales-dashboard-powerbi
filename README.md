📊 Ecommerce Sales Dashboard (Power BI)

🚀 Project Overview

This project showcases an interactive E-commerce Sales Dashboard built using Power BI.
It provides deep insights into sales performance, profitability, customer behavior, and regional trends to support data-driven decision-making.

---

📈 Key Highlights

- 💰 Total Sales Analysis (438K+ revenue)
- 📊 Profit Tracking & Performance Insights
- 📦 Quantity & Order Trends Monitoring
- 🌍 State-wise Sales Distribution
- 👥 Customer-wise Sales Analysis
- 🛒 Category & Sub-category Performance
- 💳 Payment Mode Insights (COD, UPI, Cards)

---

🛠️ Tools & Technologies

- Power BI (Data Visualization)
- Data Analysis
- Dashboard Design (UI/UX)

🔍 Business Insights

- Maharashtra generates the highest sales contribution
- COD is the most preferred payment method
- Printers and Bookcases are top profit-generating sub-categories
- Sales show fluctuations across months indicating seasonal trends

---

🎯 Objective

To transform raw e-commerce data into meaningful insights and create an intuitive dashboard for business decision-making.

---

📌 Conclusion

This dashboard enables stakeholders to quickly identify key trends, optimize sales strategies, and improve overall business performance.
